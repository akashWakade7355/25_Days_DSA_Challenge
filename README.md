# 🎯 Purpose of this Repository

This repository documents my **25-Day Data Structures & Algorithms Challenge**, where I solve DSA problems every day and upload my solutions.

The main objectives of this challenge are:

- 📚 Build a strong DSA foundation
- 💡 Improve problem-solving skills
- ⚡ Practice coding consistently
- 🎯 Prepare for coding interviews
- 🌱 Track daily learning progress
- 🤝 Help other learners by sharing clean and understandable solutions

Every solution will include:
- ✅ Topic name 
- ✅ Problem Statement
- ✅ Optimized Solution
